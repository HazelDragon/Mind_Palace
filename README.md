# Mind_Palace
Open source version of thesis 

# Requirements: 
Unity Version 2021.3.45f1
Meta Quest 2
Reciept Printer 

# How To install to Unity
1) Download zip
2) Unzip project
3) Open Unity and add project from disk

# How to change printer location
1) Open project in Unity
2) Open the Print on Throw script 
3) Change the printer name to the printer you want to print with
4) Change the filepath to the file location of where you unzipped the MemoryPrints folder within the Unity file
